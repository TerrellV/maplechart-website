# GET • pairs/

Get all pairs and their associated metadata. Use this api to browse available trading pairs.

## Query Parameters

No query parameters required

## Response

The `/pairs` endpoint will return a json document with a data property.
The data property will be an array of objects. Each object represents a pair.

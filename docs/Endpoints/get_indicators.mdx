# GET • indicators/

Get all technical indicators and their associated metadata. Use this api to browse the available indicators and their related signals.

## Query Parameters

No query parameters required

## Response

The `/indicators` endpoint will return a json document with a single data property.
The data property will be an array of objects. See below for more detail.

### JSON Schema

Metadata about the response payload

```yaml title='payload json schema'
data:
  type: array
  items:
    type: object
    properties:
      indicator_id:
        type: string
        unique: true
        description: a unique identifier representing an indicator
      description:
        type: string
        unique: false
        description: information about an indicator
      signal_ids:
        type: array
        unique: false
        description: The signal ids related to this indicator
      name:
        type: string
        unique: false
```
